# Przypomnienie podstaw programowania

Podstawowe ćwiczenia programistyczne – Python i C#

Repozytorium zawiera rozwiązania 3 prostych zadań programistycznych wykonanych jako aplikacje konsolowe w językach Python oraz C#.

## Zrealizowane zadania:
1. **Prosty kalkulator dwóch liczb**
   - pobieranie dwóch liczb i operacji (+, -, *, /)
   - wykonywanie obliczeń z użyciem instrukcji warunkowych

2. **Konwerter temperatur**
   - konwersja temperatur pomiędzy skalą Celsjusza i Fahrenheita
   - wykorzystanie instrukcji warunkowych

3. **Średnia ocen ucznia**
   - wprowadzanie wielu ocen przy użyciu pętli
   - obliczanie średniej oraz sprawdzanie warunku zaliczenia

## Wykorzystane elementy:
- zmienne i operacje arytmetyczne
- instrukcje warunkowe (`if/else`, `switch`)
- pętle (`for`, `while`)
- obsługa danych wejściowych i wyjściowych w konsoli
- podział programu na funkcje/metody
- menu umożliwiające wybór zadania

## Struktura:
- `python/` – wersja programu w Pythonie
- `csharp/` – wersja programu w C#

Program uruchamiany jest z poziomu konsoli i pozwala wybrać odpowiednie zadanie z menu.
